# Basphere 프로젝트 비전

## 프로젝트 개요

**Project Basphere**는 개발자 대상 셀프 서비스 플랫폼을 제공하는 것을 목표로 합니다.

## 핵심 목표

1. **셀프서비스**: 개발자가 인프라 팀 개입 없이 리소스 프로비저닝
2. **멀티테넌시**: 팀/프로젝트별 격리된 네트워크 환경
3. **표준화**: Golden Path를 통한 일관된 개발 환경
4. **자동화**: GitOps 기반 인프라 및 애플리케이션 배포

## 플랫폼 구성 요소

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **테넌트 네트워크** | AWS VPC와 같은 독립된 테넌트 네트워크 제공 |
| **컨테이너 플랫폼** | 셀프서비스 형식의 Kubernetes 클러스터 제공 |
| **CI/CD 플랫폼** | 플랫폼 공통 서비스 (구체적 결정 필요) |
| **코드 관리** | 플랫폼 공통 서비스 (구체적 결정 필요) |
| **컨테이너 이미지 저장소** | 플랫폼 공통 서비스 |
